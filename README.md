## 🇺🇸 English

### Overview

**Right-Click to Share** is a Windows tool that lets you share files directly from the right-click context menu.  
It copies the file to your clipboard, opens the browser automatically, and pastes it for you.

### Requirements

- Windows 10 / 11
- Must be logged in to the target SNS in your default browser

### Installation

1. Download the latest `Right-Click-to-Share.exe` from [Releases](../../releases)
2. Place it in any folder and launch it
3. Select your share targets in the settings, then click "Save & Apply to Right-Click"
4. Right-click any file → **Right-Click to Share** → Choose a destination

### How to Use

- Select **1–4 files**, right-click, and choose a share target to attach them all in one post
- Adjust the **wait time** to match your browser's startup speed (default: 3 seconds)
- Set **fixed text** to automatically insert text when posting
- You can also add **Discord** channels and **custom URLs**

### Windows 11 Users

On Windows 11, the context menu is collapsed by default, so this app's menu may not appear immediately.  
Go to the **Win11 tab** in settings and switch to **Win10 style** for direct access.

### Notes

- This app modifies the Windows registry. Use at your own risk.
- Large or long video files may fail to paste. Try increasing the wait time.
- X (Twitter) free plan has file size upload limits.

## 🇯🇵 日本語

### 概要

**Right-Click to Share** は、ファイルを右クリックするだけでSNSに共有できるWindows用ツールです。  
ファイルをクリップボードにコピーし、自動でブラウザを開いて貼り付けまで行います。

### 動作環境

- Windows 10 / 11
- 共有先SNSに事前にログインしていること

### インストール

1. [Releases](../../releases) から最新の `Right-Click-to-Share.exe` をダウンロード
2. 任意のフォルダに配置して起動
3. 設定画面でシェア先を選び、「設定を保存して右クリックに反映」をクリック
4. ファイルを右クリック → **Right-Click to Share** → シェア先を選ぶ

### 使い方

- **1〜4ファイルを選択**して右クリック → シェア先を選ぶと、まとめて1投稿に添付できます
- **待機時間**はブラウザの起動速度に合わせて調整してください（デフォルト: 3秒）
- **固定テキスト**を設定すると、投稿時に自動でテキストが入力されます
- **Discord**チャンネルや**カスタムURL**も追加できます

### Windows 11 をお使いの方

Windows 11 では右クリックメニューが折りたたまれているため、このソフトのメニューがすぐに表示されない場合があります。  
設定画面の「Win11」タブから **Win10スタイルに変更** することで、直接メニューが表示されるようになります。

### 注意事項

- 本ソフトはWindowsレジストリを書き換えます。インストール・使用は自己責任でお願いします。
- 大容量・長時間の動画ファイルは貼り付けに失敗する場合があります。待機時間を増やしてお試しください。
- X (Twitter) 無料プランはアップロードできるファイルサイズに上限があります。
